────────────────────
三、程式品質
────────────────────

檢查：

- TypeScript 型別
- 重複程式碼
- 巨型元件
- 不必要的 any
- 錯誤的 React Hook 使用方式
- Memory Leak
- 攝影機資源未釋放
- Timer 未清除
- Event Listener 未清除
- 狀態競爭
- 無限重新 Render
- Hydration Error
- Client / Server Component 使用錯誤
- 動態 Import 問題
- 無效相依套件
- 無使用套件
- Build Warning
- Console Error
- 404 資源
- 重新整理後路由失效
- 瀏覽器上一頁導致狀態錯亂

────────────────────
四、現場展示風險
────────────────────

特別模擬以下狀況：

- 無網路
- 網速慢
- 攝影機被拒絕
- 攝影機被其他程式占用
- 瀏覽器不支援 MediaPipe
- 3D 模型載入失敗
- 圖片載入失敗
- localStorage 不可用
- 使用者連續快速點擊
- 使用者中途返回
- 使用者重新整理頁面
- 全螢幕展示
- 1920 × 1080 螢幕
- 手機尺寸

核心流程在上述情況下不可以直接白屏或完全中斷。
